## Build Setup Backend 

```bash
# install dependencies
$ npm install

# serve  at localhost:3001
$ node index.js

# build for production and launch server
$ npm run build
$ npm run start
```

## Build Setup Frontend 

```bash
# install dependencies
$ npm ci install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
